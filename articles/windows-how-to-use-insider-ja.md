ときどきMicrosoft Q&A などで質問を見かける、Insider Program への参加方法や、間違って参加する人のために、Windows Insider Program への申し込み手順と最終確認画面をまとめてみました。
特に Insider Preview の参加で **もし途中まで進んで気が変わった場合、どこまでならば引き返せるのかを解説します。**

## 参加手順

現在の Insider 参加手順を示します。なお参加手順は Windows メインバージョンの更新などで時々変わる場合があるので、ご注意ください。

Windows Update の画面で「Windows Insider Program」をクリックして「Insider Program」の申し込みを始めます。


![画像](/images/windows-how-to-use-insider-ja/428135df-4e1f-404f-804b-a122d4bd6156.png)

次の「使用を開始する」確認画面です。使用を開始するをクリックして進みます。

![画像](/images/windows-how-to-use-insider-ja/0af5f56a-8d89-40c8-9cb1-92d4fd46abe1.png)

全ての環境でどうなるのかは不明ですが、私が試したPCでは、すでに予め Windows Insider アカウントへの関連付けと登録が完了していましたが、何回か試しても再度の「アカウントを紐づける」を押さないと進みませんでした。この辺りが以前から変わっているかも知れません。

![画像](/images/windows-how-to-use-insider-ja/dc2fed7f-7996-4343-a616-a3062e6824f3.png)

## 最終確認

そしていよいよお待ちかねの**最終確認画面が以下です。**きちんと最終確認をするようにとフィードバックしたのにもかかわらず、相変わらず何の変哲もない Insider チャネルの選択画面に見えます。しかしここで**「続行」をクリックして、ボタンを離した瞬間に「Insider」が確定します。
**「キャンセル」をクリックすれば、まだ引き返し出来ます。これが実質、最後です。つまり **ここで「キャンセル」をクリックして後戻りしない限り、Insider 参加が確定し、Insider 登録の Microsoft アカウントに永久に「Insider参加者」として記録されます。**

![画像](/images/windows-how-to-use-insider-ja/0758c764-4fbd-4a24-a30d-854c574eef67.png)

## 続行

「続行」をクリッした後の画面です。もう何をやっても引き返しは出来ません。指示に従うだけです。

![画像](/images/windows-how-to-use-insider-ja/7905c071-a875-4812-b711-a9a50665b733.png)

## 締めの「ようこそ」

最後の「締め？」の「Insider Program へようこそ」画面です。何回か再起動して、Insider Preview マシンが出来上がります。

![画像](/images/windows-how-to-use-insider-ja/276e72b6-571b-41c7-818f-338907689eb8.png)

ここまで来て気が変わった場合、つまり Insider Preview 評価を中止したくなったは、Windows の再インストールしかありません。

以上。
